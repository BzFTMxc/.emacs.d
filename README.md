.emacs.d
===
Emacs init

Dependencies
------------
Get `ag`
*macOS*
```
brew install the_silver_searcher
```

Usage
-----
Get `.emacs.d`
```
git clone https://github.com/BzFTMxc/.emacs.d
```
Copy/Move `.emacs.d` files to `~/.emacs.d`
```
mv .emacs.d ~/
```
Start `emacs` and wait while it installs packages
```
emacs
```
